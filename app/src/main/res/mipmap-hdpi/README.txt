# App Icons

The default Android launcher icon will be used temporarily.

When you build the project:
- Android Studio or Bitrise will use the default Android icon
- You can replace these with custom icons later

To add custom icons:
1. Create icons in these sizes:
   - mipmap-mdpi: 48x48
   - mipmap-hdpi: 72x72
   - mipmap-xhdpi: 96x96
   - mipmap-xxhdpi: 144x144
   - mipmap-xxxhdpi: 192x192

2. Name them all: ic_launcher.png

3. Place in the respective mipmap folders

For now, the app will use Android's default icon and work perfectly fine.
