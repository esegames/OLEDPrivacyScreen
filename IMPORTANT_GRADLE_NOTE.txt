WHY IS THIS MISSING?
The gradle-wrapper.jar is a binary file that needs to be downloaded from 
Gradle's servers. Since it's a binary, it's typically downloaded during 
project setup rather than included in source code.

All modern Android build systems (Android Studio, Bitrise, CI/CD platforms) 
will download this automatically when you first build the project.
